# Embed-ApScpt-faltas
Teste de uma interface (embedding) com o Google Apps Script para controle de um registro de faltas via Sheets

No Google Sheets, abra Extensões > Apps Script

Substitua o Code.gs por esse novo e salve

Rode setupInicial() uma vez, se precisar recriar as abas

Faça Implantar > Nova implantação > Aplicativo da web

Pegue a URL /exec

No GitHub, envie index.html, style.css e script.js

Ative o GitHub Pages

Abra o site publicado

Cole a URL /exec no campo do topo e clique em Carregar
